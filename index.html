Procedural Terrain

Using layered noise and DirectX I created a program in C++ to generate some procedural terrain. The program sets the textures of the terrain dynamically based on its steepness and the waterlevel, using a HLSL Shader. The shaders also allow for a number of lighting options, making use of both an ADS implementation and shadowmaps. Post processing was also implemented allowing for the generation of dynamic outlines.

Procedural Dungeon

As part of a personal project, I created a C# script to procedurally generate a dungeon in unity Using a pathfinding algorithm to ensure the dungeon was always fully explorable, it made use of premade prefabs to create a unique layout. The room prefabs could be easily created in different sizes with any number of entrances and exits, and populated with any desired objects. The program was designed to work for an online game, allowing for multiple players to explore the same dungeon online.

AI Soldiers

Starting with a framework in Unity provided by GCU, I created an AI agent making use of steering behaviours, a state machine and an A* Pathfinding algorithm. The pink agents would explore undiscovered teritory, take note of what they found, and report back to other agents. When enemy agents were found, they would group together and attack, before returning to their previous behaviour.
